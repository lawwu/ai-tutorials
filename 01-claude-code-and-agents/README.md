# Tutorial 1: Claude Code & Agent Frameworks

**Video:** https://www.youtube.com/watch?v=9jRCnYYqbJI  
**Format:** In-person, technical audience  
**Date:** March 10, 2026

## Overview

Covers foundational AI concepts through to hands-on Claude Code usage, with a focus on how agent frameworks have evolved and how to apply them in practice.

**Topics:**
- How large language models work (pretraining, tokenization, post-training)
- Evolution of agent frameworks (V1.0 LangGraph-style → V2.0 CLI agents)
- Claude Code basics and hands-on demo
- Skills and plugins (agentskills.io, knowledge-work-plugins)
- Harness engineering (chaining skills together)
- Agentic knowledge work mindset
